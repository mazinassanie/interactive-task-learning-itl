# Interactive Task Learning (ITL) Framework 🗣️🤖

[![License: MIT](https://img.shields.io/badge/License-MIT-orange.svg)](https://opensource.org/licenses/MIT)

A framework for building autonomous agents capable of learning new tasks through natural interaction and instruction.

## 🚀 Overview
ITL allows human instructors to teach agents new procedures using natural language or demonstration. This repository focuses on the cognitive mechanisms required to map instructions to internal procedural memory and working memory structures.

## ✨ Key Features
- **Semantic Instruction Parsing**: Decodes verbal/textual instructions into goal-oriented representations.
- **Task Encoding**: Dynamically creates new rules in procedural memory for newly learned tasks.
- **Cross-Domain Learning**: Adaptable across different environments (robotics, game AI, virtual assistants).

## 💻 Usage
```python
agent = ITLAgent()
agent.process_instruction("Pick up the red ball")
```

## 📚 Background
This work is based on research in **Interactive Task Learning** and **Cognitive Architectures**, focusing on making AI agents more flexible and easier to collaborate with.
